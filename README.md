# To-Do app 
A simple MVVM architecture based To-Do application build by following [Coding in Flow](https://www.youtube.com/watch?v=Udk6iaR-RXA&list=PLrnPJCHvNZuCfAe7QK2BoMPkv2TGM_b0E) tutorial.

### Library Used
- Room 
- Dagger Hilt 
- Coroutines
- Navigation Component
